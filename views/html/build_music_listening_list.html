Modify Existing Music

To modify existing music using the form below, enter a Music Listening List ID shown on the right corresponding to the list item you would like to modify. Then, also enter a band name, album name, and release year for that list item in the form below. You can choose to update any number of those three fields with new data. Be sure to enter data for all three fields. After submitting the form below, click the Get Music Listening List button to refresh and view the updated list of music data. This new list will include the modification for the list item with the ID you specified in the form below.
